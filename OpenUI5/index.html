Step by step

  • Свалете и инсталирайте Git Bash
  • Акаунт на HCP
  • Влезте в акаунта си
  • В менюто Content->HTML5 Applications
  • New Application..
  • Дайте име на приложението

Step by step

  • Кликнете върху името на приложението
  • От менюто(ляво)->Development
  • Копирайте Git Repository URL
  • Отворете папката, в която искате да е проекта ви
  • Отворете инстанция на Git Bash

Step by step

  • git clone RepoURL
  • Сложете проекта си в новосъздадената папка
  • git config user.email 'mail@mail.com'
  • Work work ..

Push to HANA..

  • git add .
  • git commit -m 'some message'
  • git push
  • Иска мейл и парола от HANA ondemand

Version management

  • Вървете на HANA ondemand
  • Кликнете върху създадения проект
  • Content->Development
  • Create Version
  • Version Management->Activate Version
  • Отворете Application URL
